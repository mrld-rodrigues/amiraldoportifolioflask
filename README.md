<h1># Meu Portfólio</h1>

Este é um projeto de desenvolvimento de um portfólio utilizando as tecnologias Flask, Python, Html, Css, Javascript e o Bootstrap. O objetivo deste projeto é criar um site pessoal para apresentar informações, projetos, habilidades e formas de contato.

<h2>## Funcionalidades</h2>

- Apresentação pessoal
- Exibição de projetos desenvolvidos
- Listagem de habilidades e tecnologias
- Formulário de contato

<h2>## Tecnologias Utilizadas</h2>

- Flask
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Python
